Mes projets

Projets académiques


Création d'un site web institutionnel d'une entreprise

Naufrage du Titanic

Outils utilisés :

  • Word, Excel et internet
  • Lucidchart
  • Visual Studio Code
  • SQL

Le but de ce projet a été la création et l’implémentation d’une base de données afin de voir les facteurs ayant influencé la survie des passagers du Titanic ayant fait naufrage dans la nuit du dimanche 14 avril 1912. Ce projet à donc été découpé en 3 parties.

Dans la première partie, nous avons dû rédiger un document contenant une présentation synthétique de l’organisation du sauvetage des passagers et la justification de l’impossibilité logistique d’envisager de sauver plus de passagers. Nous avons aussi dû créer des diagrammes SEA et rédigé un ensemble de règles pour la suite du projet.

Dans la deuxième partie, nous avons dû rédiger un schéma du modèle relationnel dérivé du SEA précédemment rédigé. Nous avons aussi codé en SQL la création et suppression de notre base de données en respectant les schémas et les règles que nous avions produits dans la première partie.

Dans la dernière partie, nous avons interrogé notre base de données afin de pouvoir en déduire des différents facteurs ayant influencé la survie des passagers du Titanic.
